# CardTitle

## Component representing the title of a card

`CardTitle` is used when creating the title of a card, **you don't need to use this component directly**, as a title is passed by props to a `Card`component.

`title` should always be present when using `CardTitle`.

The width of `CardTitle`component is limited so as to avoid extremely long titles that can break `Card`component. It's width is controlled via CSS, showing an ellipsis when text is longer than max.

```css
.card-title {
  max-width: 20rem;
  overflow: hidden;
  white-space: nowrap;
  text-overflow: ellipsis;
  text-transform: capitalize;
  font-size: 1.5rem;
}
```
