# ScrollBox

## a component representing a group of cards

`ScrollBox` is used when creating a list of cards, each one representing a group of gifts. **This is the component you will use**.

### Required props

1. `scrollBoxTitle`, a string representing the title of a `ScrollBox` component.

2. `loading` a boolean representing the loading state of component.

3. `cards` an array of cards, its type being an intersection of `CardType` and `{ id: string }` as React requires a unique `key` prop when rendering a list of components. A card has this shape:

```javascript
[{
  id: string,
  title: string,
  desc: string,
  tags: [ { title: string}, { title: string}, ...],
  }]
```

## `ScrollBox` component can be in three different states:

export const Template = (args) => <ScrollBox {...args} />;
