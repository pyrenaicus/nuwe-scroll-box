# CardDescription

## Component representing the description of a card

`CardDescription` is used when creating the description of a card, **you don't need to use this component directly**, as a description is passed by props to a `Card`component.

`desc` should always be present when using `CardDescription`.

export const Template = (args) => <CardDescription {...args} />;
