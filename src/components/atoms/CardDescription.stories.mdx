# CardDescription

## Component representing the description of a card

`CardDescription` is used when creating the description of a card, **you don't need to use this component directly**, as a description is passed by props to a `Card`component.

`desc` should always be present when using `CardDescription`.

The width and height of `CardDescription`component is limited so as to avoid extremely long descriptions that can break `Card`component. It's height is **limited to 2 lines**, width and height are controlled via CSS, showing an ellipsis when text is longer than max.

```css
.card-description {
  max-width: 20rem;
  overflow: hidden;
  display: -webkit-box;
  line-clamp: 2;
  -webkit-line-clamp: 2;
  -webkit-box-orient: vertical;
}
```
