# TagList component

## A component representing a list of tags

`TagList` is used when creating a group of tags, **you don't need to use this component directly**, as tags are passed by props to a `Card`component.

An array of tags with shape `[{ title: string }, { title: string }, ...]` should always be present when using `TagList`.

export const Template = (args) => <TagList {...args} />;
