# Card

## A component representing a single card

`Card` is used when creating a single card, to show a group of gifts, **you don't need to use this component directly**, as cards are passed by props to a `ScrollBox`component.

`tags`, an array of tags with shape `[{ title: string }, { title: string }, ...]` should always be present when using `Card`.

`title`, a string, should always be present when using `Card`.

`desc`, a string, should always be present when using `Card`.

export const Template = (args) => <Card {...args} />;
